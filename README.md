# Diploma Thesis API

A Django-based backend API developed for a diploma thesis project, focused on logistics and fleet management.

## Features
- **Truck Management:** Tracking and managing vehicle data.
- **GPS Integration:** Real-time or logged location services.
- **Chat System:** Communication module for users.
- **Admin Panel:** Custom management interface.
- **Media Points:** Handling location-based media or points of interest.
- **User Work Tracking:** Monitoring user activities and tasks.

## Tech Stack
- **Language:** Python
- **Framework:** Django
- **Frontend:** HTML (Admin/Basic Views)
